# Matteo Loglio
[Matteo Loglio](https://matlo.me) Matteo Loglio is a designer and artist, currently experimenting with the creative applications of artificial intelligence. Recently he designed a musical synthesizer for Google, he taught a river in London to write poetry and started the ed-tech company [Cubetto](https://primotoys.com): a wooden robot that is changing the lives of the millions of children using it every day. Matteo talks about design and creativity in universities, museums and hackspaces around the world. Sometimes you can find his work exhibited in places like MoMA NY, V&A London, or Triennale Milan
